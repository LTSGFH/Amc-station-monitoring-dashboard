# AMC Station Monitoring Dashboard (Power BI)

## Project Overview
This project monitors AMC stations and engineer workload using Power BI.  
The dashboard helps track station activity, engineer allocation, and operational performance.

## Tools Used
- Power BI
- Excel
- DAX
- Data Cleaning

## Dashboard Features
- Total stations monitored
- Engineer workload analysis
- Station activity tracking
- Operational performance metrics

## Key Insights
- Workload distribution across engineers
- Active vs inactive stations
- Operational monitoring for AMC services
